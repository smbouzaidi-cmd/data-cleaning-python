# Data Cleaning Python Script
Python script to clean and organize CSV/Excel data for analysis.
## Purpose
This script helps you remove duplicates, handle missing values, and fix dates in CSV files.
## Requirements
- Python 3.x
- pandas library
- ## Usage
1. Place your CSV file in the same folder as the script.
2. Update the filename in the script if necessary.
3. Run the script: `python data_cleaning.py`
4. The cleaned data will be saved as `cleaned_sales.csv`
5. ## Note
This script can be reused for any CSV or Excel file by changing the filename and column names.
